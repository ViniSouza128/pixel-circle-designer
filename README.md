# 🎯 Pixel Circle / Ellipse Generator

Interactive tool for generating pixel-perfect circles and ellipses with real-time visualization.

🌐 Live Demo:
https://vinisouza128.github.io/pixel-circle-designer/

---

## 📌 About the Project

This project is an interactive pixel circle and ellipse generator, ideal for:

- 🎮 Minecraft builds
- 🎨 Pixel art creation
- 🧩 Grid-based design
- 🧠 Graphics algorithm exploration

---

## ⚙️ Features

- ✔ Real-time circle and ellipse generation
- ✔ Rendering modes:
  - Filled
  - Thin
  - Thick
- ✔ Available algorithms:
  - Euclidean Distance
  - Bresenham (Midpoint)
  - Threshold
- ✔ Adjustable size, width, and height
- ✔ Light/Dark mode toggle
- ✔ PNG export
- ✔ Grid and center guide visualization

---

## 🧠 Algorithms

The project implements multiple rasterization approaches:

- Euclidean Distance — continuous math-based calculation
- Bresenham — classic efficient integer-based algorithm
- Threshold — coverage-based approximation
